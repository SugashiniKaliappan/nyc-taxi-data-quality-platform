# NYC Taxi Data Quality Platform

A Data Engineering project for automated data quality monitoring on NYC Taxi datasets.

## Features

- Data Profiling
- Rule-Based Quality Validation
- Anomaly Detection
- Data Drift Detection
- Automated GitHub Actions Pipeline

## Tech Stack

- Python
- Pandas
- NumPy
- GitHub Actions
- Parquet

## Dataset

NYC TLC Yellow Taxi Trip Records

## Pipeline

1. Data Profiling
2. Quality Checks
3. Anomaly Detection
4. Drift Detection
5. Report Generation

## Sample Results

- Quality Score: 98.91%
- Records Processed: 2.96 Million
- Drift Detection: January vs February
